Generate Settings File

Enter all the information and click on "Generate Settings" button to generate the "deployment.settings" file. After download you can customize the file name. Run the script to deploy a vm with option "-file". The "-file" is the path of "deployment.settings" file generated. For more details on instructions and examples on how to deploy vm using this setting file, please refer the Deployment Documentation

Note: Mandatory to use either IPv4 or IPv6 or Both. Do not leave both with default values.
